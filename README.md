# Squirrel Notes

An interactive pixel art website featuring a character in Lincoln Park, Chicago. Users click "Place Walnut" to trigger an animation sequence where a squirrel delivers notes.

## Project Structure

```
squirrel-notes/
├── index.html
├── styles.css
├── script.js
├── assets/
│   ├── sprites/
│   │   ├── character/
│   │   └── squirrels/
│   ├── backgrounds/
│   └── audio/
└── README.md
```

## Development

Built with vanilla HTML, CSS, and JavaScript. No frameworks required.

## Deployment

Hosted on Vercel, connected to this GitHub repository.
